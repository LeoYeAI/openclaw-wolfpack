**Wolfpack** es un Skill multi-agente para [OpenClaw](https://docs.openclaw.ai). Despliega **8 agentes IA especializados** como equipo coordinado con puertas de calidad SHARP y 4 plantillas de flujo de trabajo.

### 🐺 La Manada

| Agente | Rol | Responsabilidad |
|--------|-----|----------------|
| 🧠 Planner | Director | Descomposición de tareas, coordinación |
| 💡 Ideator | Motor creativo | Generación de ideas, evaluación de novedad |
| 🎯 Critic | Control de calidad | Evaluación SHARP (≥18 para pasar) |
| 📚 Surveyor | Experto en literatura | Búsqueda bibliográfica |
| 💻 Coder | Ingeniero | Implementación, experimentos |
| ✍️ Writer | Redactor | Escritura académica, LaTeX |
| 🔍 Reviewer | Evaluador | Revisión por pares simulada |
| 📰 Scout | Explorador | Resumen diario de papers |

### Inicio rápido

```bash
git clone https://github.com/LeoYeAI/openclaw-wolfpack.git
cd openclaw-wolfpack && chmod +x scripts/setup.sh
./scripts/setup.sh --mode local --model anthropic/claude-sonnet-4-5
```
