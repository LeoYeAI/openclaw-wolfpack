**Wolfpack** ist ein Multi-Agenten-Skill für [OpenClaw](https://docs.openclaw.ai). Es deployt **8 spezialisierte KI-Agenten** als koordiniertes Team mit SHARP-Qualitätstoren und 4 Workflow-Vorlagen.

### 🐺 Das Rudel

| Agent | Rolle | Verantwortung |
|-------|-------|---------------|
| 🧠 Planner | Dirigent | Aufgabenzerlegung, Koordination |
| 💡 Ideator | Kreativmotor | Ideengenerierung, Neuheitsbewertung |
| 🎯 Critic | Qualitätstor | SHARP-Bewertung (≥18 zum Bestehen) |
| 📚 Surveyor | Literaturexperte | Literaturrecherche |
| 💻 Coder | Ingenieur | Implementierung, Experimente |
| ✍️ Writer | Autor | Akademisches Schreiben, LaTeX |
| 🔍 Reviewer | Gutachter | Simuliertes Peer-Review |
| 📰 Scout | Aufklärer | Täglicher Paper-Digest |

### Schnellstart

```bash
git clone https://github.com/LeoYeAI/openclaw-wolfpack.git
cd openclaw-wolfpack && chmod +x scripts/setup.sh
./scripts/setup.sh --mode local --model anthropic/claude-sonnet-4-5
```
