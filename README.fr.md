**Wolfpack** est un Skill multi-agents pour [OpenClaw](https://docs.openclaw.ai). Il déploie **8 agents IA spécialisés** en équipe coordonnée avec des contrôles de qualité SHARP et 4 modèles de workflow.

### 🐺 La Meute

| Agent | Rôle | Responsabilité |
|-------|------|----------------|
| 🧠 Planner | Chef d'orchestre | Décomposition des tâches, coordination |
| 💡 Ideator | Moteur créatif | Génération d'idées, évaluation de nouveauté |
| 🎯 Critic | Contrôle qualité | Évaluation SHARP (≥18 pour passer) |
| 📚 Surveyor | Expert littérature | Recherche bibliographique |
| 💻 Coder | Ingénieur | Implémentation, expériences |
| ✍️ Writer | Rédacteur | Rédaction académique, LaTeX |
| 🔍 Reviewer | Évaluateur | Revue par les pairs simulée |
| 📰 Scout | Veilleur | Digest quotidien, tendances |

### Démarrage rapide

```bash
git clone https://github.com/LeoYeAI/openclaw-wolfpack.git
cd openclaw-wolfpack && chmod +x scripts/setup.sh
./scripts/setup.sh --mode local --model anthropic/claude-sonnet-4-5
```
