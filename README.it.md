**Wolfpack** è uno Skill multi-agente per [OpenClaw](https://docs.openclaw.ai). Distribuisce **8 agenti IA specializzati** come team coordinato con gate di qualità SHARP e 4 modelli di workflow.

### 🐺 Il Branco

| Agente | Ruolo | Responsabilità |
|--------|-------|----------------|
| 🧠 Planner | Direttore | Scomposizione compiti, coordinamento |
| 💡 Ideator | Motore creativo | Generazione idee, valutazione novità |
| 🎯 Critic | Controllo qualità | Valutazione SHARP (≥18 per passare) |
| 📚 Surveyor | Esperto letteratura | Ricerca bibliografica |
| 💻 Coder | Ingegnere | Implementazione, esperimenti |
| ✍️ Writer | Autore | Scrittura accademica, LaTeX |
| 🔍 Reviewer | Revisore | Revisione paritaria simulata |
| 📰 Scout | Esploratore | Digest giornaliero dei paper |

### Avvio rapido

```bash
git clone https://github.com/LeoYeAI/openclaw-wolfpack.git
cd openclaw-wolfpack && chmod +x scripts/setup.sh
./scripts/setup.sh --mode local --model anthropic/claude-sonnet-4-5
```
